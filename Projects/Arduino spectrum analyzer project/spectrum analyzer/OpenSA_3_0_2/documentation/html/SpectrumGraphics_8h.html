Sets pixel value with gamma correction.

Bounds checking is performed. color values are mapped to display brightness values using the COLORS array. The number of supported colors is defined by NCOLORS. Results are drawn to the drawing buffer and are not immediately available on screen. Call show() to make the results of drawing available on screen.

Parameters:
rrow
ccolumn
vvalue between 0 and NCOLORS-1
